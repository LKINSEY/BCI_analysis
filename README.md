# BCI_analysis

### Install
To use as a package: 
- clone from github 
```
git clone https://github.com/kpdaie/BCI_analysis.git
```
- enter to the root directory of this repo 
```
cd BCI_analysis
```
- and pip install the package:
```
pip install -e .
```


### File Structure 
```
|- BCI_analysis
    |- io
    |- plot
    |- utils
|- Notebooks
```
